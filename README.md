# UniPulse Indicator (MQL4)

UniPulse is a trend-following indicator for MetaTrader 4 that filters 
out low-volatility noise using a volatility-based confirmation engine.

## Features
- Configurable signal quality controls
- Optional candle-body confirmation
- Multi-channel alert system (popup, push, e-mail)
- Strictly non-repainting — all signals are calculated exclusively 
  on closed candles

## Installation
1. Copy `UniPulse.mq4` into your MetaTrader 4 `MQL4/Indicators` folder
2. Restart MT4 or refresh the Navigator panel
3. Drag the indicator onto your chart
